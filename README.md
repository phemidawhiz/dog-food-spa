
## Dog Food Single Page Application (SPA) Design

This project was build with latest version of next.js, tailwind css and typescript using a component based approach.

To run on your local machine, ensure you have node js and npm install and follow the instructions below

First, install dependencies with:

```bash
npm install
# or
yarn
```

Secondly, run the SPA with the commands:

```bash
npm run dev
# or
yarn dev
```

The project should be visible on your local machine at the address http://localhost:3000
